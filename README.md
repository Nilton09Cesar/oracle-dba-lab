# oracle-dba-lab

Este repositório documenta minha jornada prática na preparação para a certificação Oracle Database Administration I (1Z0-082), com foco no desenvolvimento de habilidades essenciais para atuação como DBA Oracle Júnior/pleno.
O objetivo é consolidar conhecimentos técnicos através de laboratórios práticos, simulações reais e documentação estruturada, seguindo boas práticas de administração de banco de dados Oracle.

• Objetivos
Dominar fundamentos de administração Oracle
Executar tarefas reais de DBA em ambiente controlado
Criar portfólio técnico validado
Preparar para certificação Oracle

Tecnologias Utilizadas
Oracle Database 19c
SQL*Plus
SQLcl
Oracle Linux 7.9 
Bash scripting
Git e GitHub
